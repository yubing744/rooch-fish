# RoochFish

“你鱼了吗”是一个基于 Rooch 区块链的开源全链游戏，玩家可以在链上鱼塘中购买和控制鱼，通过吃掉其他鱼和食物来成长，并在特定条件下获得代币奖励。该项目的智能合约部分是开源的，旨在为开发者和玩家提供一个透明和可扩展的游戏环境。

## 项目简介

- **游戏名称**：你鱼了吗
- **区块链平台**：Rooch
- **智能合约语言**：Move
- **游戏玩法**：
  - 玩家使用 RGAS 代币购买鱼。
  - 鱼在鱼塘中随机生成位置并赋予固定初始大小。
  - 玩家可以控制鱼的移动，吃掉比自己小的鱼和食物来成长。
  - 玩家可以向鱼塘投喂食物，并在鱼销毁时获得分成。
  - 鱼可以移动到出口位置销毁，玩家根据鱼的大小获得 RGAS 代币奖励。

## 快速开始

### 环境准备

1. **安装 Move 开发工具**：
   - 请确保您的系统上已安装 Move 语言的编译器和相关工具。

2. **克隆仓库**：
   ```bash
   git clone https://github.com/yubing744/rooch-fish.git
   cd rooch-fish
   ```

### 部署合约

1. **编译合约**：
   - 使用 Rooch Move 编译器编译合约代码。
   ```bash
   rooch move build
   ```

2. **部署合约**：
   - 将编译后的合约部署到 Rooch 区块链上。
   - 请参考 Rooch 的官方文档进行合约部署。

### 参与游戏

- 玩家可以通过游戏界面（`rooch-fish-web` 项目）参与游戏。
- 通过界面购买鱼、控制鱼的移动、投喂食物并赚取代币。
